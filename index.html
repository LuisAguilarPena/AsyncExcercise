Promises Challenge

Using Javascript's Fetch API and the following URL https://source.unsplash.com/random/ create the following functions:

1. A function called apiPending that makes a request to the URL above and returns a Promise object. When logging the function call it should show a Promise in the pending state.

2. A function called apiPromise that replicates the previous one and consumes the promise scheduling a callback when the promise has resolved into either fulfilled or rejected. The callbacks should log the response on fulfillment or on rejection. Regardless of the response's outcome log the following message "will execute regardless!".

3. A function called apiAsync that uses the keywords async & await that logs the following: "apiAsync response is:" followed by the actual response if the Promise is fulfilled and "apiAsync error is:" followed by the actual error if the Promise is rejected. Finally return a response.

4. A function called apiSearch that accepts any number of keywords as arguments that fetches a random search term from https://source.unsplash.com/ and returns a promise object.

5. A function that waits for both apiSearch and apiSearch and adds both images to the HTML document.
